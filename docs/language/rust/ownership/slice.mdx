## String Slice

A string slice is a <span style={{color: "var(--secondary-font-color)"}}> reference </span> to part of a `String`:

``` rust lines
let s = String::from("hello world");

let hello: &str = &s[0..5]; // exclusive upper bound
let world = &s[6..11];
```

The slice data structure stores the <span style={{color: "var(--secondary-font-color)"}}> starting position </span> and the <span style={{color: "var(--secondary-font-color)"}}> length </span> of the slice

In the case of `let world = &s[6..11];`, `world` would be a slice that contains a pointer to the byte at index 6 of `s` with a length value of 5.

![slice](../assets/slice.svg)

<Callout type="warning">

String slice range indices must occur at valid UTF-8 character boundaries.

If you attempt to create a string slice in the middle of a multibyte character, your program will exit with an error.

</Callout>

### Syntax short hand

#### First char of String

``` rust lines
let s = String::from("hello");

let slice = &s[0..2];
let slice = &s[..2];
```

#### Last char of String

``` rust lines
let s = String::from("hello");

let len = s.len();

let slice = &s[3..len];
let slice = &s[3..];
```

#### Entire string slice

``` rust lines
let slice = &s[..];
let slice = &s;
```

## String Literals as Slices

string literals being stored inside the binary.

``` rust
let s = "Hello, world!";
```

The type of `s` here is `&str`: <span style={{color: "var(--secondary-font-color)"}}> it's a slice </span> pointing to that specific point of the binary.

<Callout type="info">

Type `&String` is compatible with `&str`

</Callout>

## Miscellaneous

string to bytes array:

``` rust lines
let s = String::from("foo");
let bytes = s.as_bytes();
```

set `String` to empty (`""`):

``` rust
<String>.clear();
```

``` rust lines
fn slice_out_of_array() {
    let a = [1, 2, 3, 4, 5];

    let nice_slice = &a[1..=3];

    assert_eq!([2, 3, 4], nice_slice)
}

```
