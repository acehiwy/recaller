A key-value data structure

## Create

``` python lines
def foo():
  return 'foo'

my_dict = { 
  "asdf" : 123 , 
  2222 : "sdfhs" , 
  (1,2) : "2222", 
  foo: "233"
}
```

<Callout type="note"> 

Dictionaries are indexed by keys, which can be any <span style={{color: "var(--secondary-font-color)"}}> immutable </span> type

</Callout>

<Callout type="note"> 

Dictionaries order are preserve as the insertion order

</Callout>

<Callout type="note"> 

Tuples can be used as keys if they contain only strings, numbers, or tuples

</Callout>

### Dictionaries comprehensions

``` python lines
{x: x**2 for x in (2, 4, 6)}
# {2: 4, 4: 16, 6: 36}
```

### Create from sequences <span style={{color: "var(--primary-font-color)"}}> of key-value pairs </span>

``` python lines
# create from key-value list
dict( [('sape', 4139), ('guido', 4127), ('jack', 4098)] )
# {'sape': 4139, 'guido': 4127, 'jack': 4098}

# create from key-value tuple
dict( ((1,2), (3,4)) )
# {1: 2, 3: 4}
```

### <span style={{color: "var(--not-nesary-for-now-future-font-color)"}}> Create using keyword argument </span>

``` python lines
dict(sape=4139, guido=4127, jack=4098)
# {'sape': 4139, 'guido': 4127, 'jack': 4098}
```
## Read value

``` python lines
def foo():
  return 'foo'

my_dict = { 
  "asdf" : 123 , 
  2222 : "sdfhs" , 
  (1,2) : "2222", 
  foo: "233"
}

some_var = 2222
my_dict["asdf"]
my_dict[some_var]
my_dict[(1,2)]
my_dict[foo]
```

## Write value

``` python lines
my_dict["asdf"] = 'something_else'
```

## Check for existing key

``` python lines
def foo():
  return 'foo'

my_dict = { 
  "asdf" : 123 , 
  2222 : "sdfhs" , 
  (1,2) : "2222", 
  foo: "233"
}

some_var = 2222
foo in my_dict
# True

some_var in my_dict
# True

(1,2) in my_dict
# True

'else' not in my_dict
# False
```

## Delete key

``` python lines
del foo[some_key]
```


## <span style={{color: "var(--not-nesary-for-now-future-font-color)"}}> Extract dictionaries' key as list </span>

``` python lines
my_dict = list({1: "foo", "2" : "bar" , 3: "baz"})
# [1, '2', 3]
```
