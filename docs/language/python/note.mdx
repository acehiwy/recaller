<span style={{color: "var(--can-not-understand-font-color)"}}> Some Python modules are also useful as scripts. You can invoke them with `python -m module [arg] ...`, which executes the module's source file as if you had spelled out its full name on the command line. </span>

## Passing argument to the script

Script arguments can be accessed with

``` python lines
import sys

sys.argv # list of string
```

The length of the list is at least one.

When no script and no arguments are given, `sys.argv[0]` is an empty string.

When the script name is given as `-` (meaning standard input), `sys.argv[0]` is set to `-`

In interactive mode, the last printed expression is assigned to the variable `_`

``` python
2+3
# 5
_ * 2
# 10
_ + 1
# 11
```

<span style={{color: "var(--can-not-understand-font-color)"}}> When `-m module` is used, `sys.argv[0]` is set to the full name of the located module. Options found after `-c command` or `-m module` are not consumed by the Python interpreter's option processing; they are left in `sys.argv` for the command or module to handle. </span>

## <span style={{color: "var(--not-require-but-good-to-know-font-color)"}}> Source Code Encoding </span>

By default, Python source files are treated as encoded in UTF-8.

declaring other encoding other than default one use special comment should be added as the first line of the file

``` python title="foo.py" lines
# -*- coding: <encoding> -*-
```

``` python title="foo.py" lines
# -*- coding: cp1252 -*-
```

``` bash title="foo.sh" lines
#!/usr/bin/env python3
# -*- coding: cp1252 -*-
```



where `<encoding>` is one of [codecs](https://docs.python.org/3/library/codecs.html#module-codecs)

## Miscellaneous

Division `/` always returns a float. To do floor division and get an integer result you can use the `//` operator;

``` python
1273/3
# 424.3333333333333
1273//3
# 424
```

Simple assignment in Python never copies data
``` python lines
var1 = "foo"
var2 = "foo"
var1 == var2
# True
id(var1) == id(var2)
# True
```

`print()` accept `end=` argument for specifying end of the line characters
``` python lines
print("foo", end="something not new line")
```
