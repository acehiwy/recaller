# Tuple

Tuples are <Color color="var(--primary-font-color)"> immutable </Color> sequences

```python title= showLineNumbers
t = (12345, 54321, 'hello!')  # or declare without (). eg, t = 12345, 54321, 'hello!'
t[0]
# 12345
t
# (12345, 54321, 'hello!')


# Tuples are immutable:
t[0] = 88888
#  highlight-error-next-line
# TypeError: 'tuple' object does not support item assignment
```

Tuples may be nested:
```python title= showLineNumbers
u = ( t, (1, 2, 3, 4, 5) )
u
# ((12345, 54321, 'hello!'), (1, 2, 3, 4, 5))
```

<Admonition type="caution" title="caution"> 

Single item tuple is constructed with trailing comma, `,`

```python title= showLineNumbers
this_is_interpreted_as_string = ("hello")

this_is_tuple = ("hello",) # notice trailing comma
this_is_also_tuple = "hello", # notice trailing comma
```

</Admonition>

<Admonition type="info" title="info"> 

Empty tuple syntax: `empty = ()`

</Admonition>


## Tuple compreshension

```python title= showLineNumbers
my_tuple = tuple( x + 2 for x in [1,2,3,4] )
```


## `namedtuple`

It is like tuple additionally each item can be access by name

```python title= showLineNumbers
from collections import namedtuple

MyTuple = namedtuple('MyTuple' , ['key1' , 'key2'])
print(MyTuple)
# <class '__main__.MyTuple'>

my_tuple_instance = MyTuple("key1_value" , key2="key2_value")

print(my_tuple_instance.key1 , my_tuple_instance.key2)
# key1_value key2_value

print(my_tuple_instance[0] , my_tuple_instance[1])
# key1_value key2_value
```

2nd argument of `namedtuple`, `field_name` can be
- list of string. `["key1", "key2"]`
- a string with white spaces. `"key1 key2"`
- a string with commas. `"key1, key2"`

each field name need to be a valid string that can used to crate attribute name

<Admonition type="note" title="note"> 

Field name can not start with underscore, `_`. 

</Admonition>

<Admonition type="info" title="info"> 

`namedtuple` is a subclass of tuple.

</Admonition>

### Methods

#### `_make`

create `namedtuple` instance from iterable

```python title= showLineNumbers
MyTuple = namedtuple('MyTuple' , ['key1' , 'key2'])

ins = MyTuple._make(['key1_value', 'key2_value'])
# MyTuple(key1='key1_value', key2='key2_value')
```

#### `_asdict`

return dict representing tuple

```python title= showLineNumbers
MyTuple = namedtuple('MyTuple' , ['key1' , 'key2'])

ins = MyTuple('key1_value', 'key2_value')
# highlight-next-line
ins._asdict()
# {'key1': 'key1_value', 'key2': 'key2_value'}
```
