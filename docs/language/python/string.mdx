``` python lines
some_string = "string value"
```

double quote and single quote are equivalent

escape character is `\`
``` python lines
>>> 'doesn\'t'  # use \' to escape the single quote...
"doesn't"
```

with `print()`, special characters are interpreted

``` python lines
s = 'First line.\n Second line.' 
s  # without print(), special characters are included in the string
# 'First line.\n Second line.'

print(s)  
# First line.
# Second line.
```

<Info> 

use raw string if you don't want characters prefaced by \ to be interpreted as special characters

``` python lines
print(r'C:\some\name')  # note the r before the quote
# C:\some\name
```

</Info>

## Multiple line string 

use triple quote:
``` python lines
print("""\
Usage: thingy [OPTIONS]
     -h                        Display this usage message
     -H hostname               Hostname to connect to
""")
```

<Note> 

End-of-line characters are automatically <span style={{color: "var(--secondary-font-color)"}}> included on each line </span>, but you can prevent that by adding `\` at the end of the line.

</Note>

## Concat string

``` python lines
3 * 'un' + 'ium'
# 'unununium'
```

Two or more <span style={{color: "var(--primary-font-color)"}}> string literals </span> next to each other are automatically concatenated.

``` python lines
'foo' 'bar'  'baz'
# 'foobarbaz'
```

## String indexing

``` python lines
word = 'Python'
word[0]  # character in position 0
# 'P'
word[5]  # character in position 5
# 'n'

word[-2]  # second-last character
# 'o'
word[-6]
# 'P'
```

## Slice (substring)

``` python lines
word = 'Python'
word[0:2]  # characters from position 0 (included) to 2 (excluded)
# 'Py'

word[:2]   # character from the beginning to position 2 (excluded)
# 'Py'
word[4:]   # characters from position 4 (included) to the end
# 'on'
word[-2:]  # characters from the second-last (included) to the end
# 'on'
```

<Note> 

Index substitution, the start is always included, and the end always excluded

</Note>

<Note> 

out of range slice indexes are handled gracefully:

``` python lines
word = "python"
word[4:42]
# 'on'
word[42:]
# ''
```
</Note>

use `len()`

## String interpolation

``` python lines
result = f'{var_1} is not the same as {var_2}'
```

<Info> 

`str.format()` can be used alternatively. See more info [here](https://docs.python.org/3/library/stdtypes.html#str.format)

</Info>

<Info> 

The `{}` syntax can do much more than substitute a string. It can format a string in almost any way you like.

see more info [here](https://docs.python.org/3/reference/lexical_analysis.html#f-strings)

</Info>

## Miscellaneous

Use [`str()`](https://docs.python.org/3/library/stdtypes.html#str) or [`repr()`](https://docs.python.org/3/library/functions.html#repr) function to view a string representation of an object.

Convert string to byte, [`str.encode()`](https://docs.python.org/3/library/stdtypes.html#str.encode)

JSON serializing and deserializing
``` python lines
import json
x = [1, 'simple', 'list']

# serializing
json.dumps(x)


# deserializing
json_string = '...'
json.loads(json_string)
```
