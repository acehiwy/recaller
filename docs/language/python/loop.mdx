``` python lines
for i in range(5): # or range(2,5) -> 2,3,4
    print(i)
```

<Accordion title="output">

``` bash lines
# 0
# 1
# 2
# 3
# 4
```

</Accordion>

<Info> 

The `range()` function returns an [iterable](https://docs.python.org/3/glossary.html#term-iterable).

</Info>



## Loop on a list

``` python lines
words = ['cat', 'window', 'defenestrate']
for w in words:
    print(w, len(w))
```

Get a list item's index.

``` python lines
words = ['cat', 'window', 'defenestrate']
for index, value in enumerate(words):
    print(index, value)
```

## Loop on dict

``` python lines
users = {'Hans': 'active', 'Éléonore': 'inactive', '景太郎': 'active'}

for key, value in users.copy().items():
    # do something
```

<Warning> 

Modifying and iterating over the same collection should be avoided. Use a copy instead, for example `<collection>.copy()`.

</Warning>

### `break` and `continue`


``` python lines highlight={3,6}
for ... :
  # do something
  continue
  for ... :
    # do something
    break
```

The `break` statement exits the <span style={{color: "var(--secondary-font-color)"}}> innermost enclosing </span> `for` or `while` loop.

---

### `else` on Loop


If the loop <span style={{color: "var(--secondary-font-color)"}}> finishes without </span> executing the `break`, the `else` clause executes.

``` python lines
for ... :
  # do something
else:
  print("print if no break")
```

``` python lines
i = 0
while i <= 3:
  print(i)
  i = i +1 
  break
else:
  print('print if no break')

# output
# 0
```
