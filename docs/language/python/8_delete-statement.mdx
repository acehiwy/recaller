# `del` statement

use to delete item(s) from a list or delete a variable

```python title= showLineNumbers
a = [-1, 1, 66.25, 333, 333, 1234.5]
del a[0]
# [1, 66.25, 333, 333, 1234.5]
del a[2:4]
# [1, 66.25, 1234.5]
del a[:]
# []
```

delete variable

```python title= showLineNumbers
del a
```

<Note> 

 the statement del x removes the binding of x from the namespace referenced by the local scope

</Note>


<Info> 

More info [here](https://docs.python.org/3/reference/simple_stmts.html#del)

</Info>

<Info>

</Info>
