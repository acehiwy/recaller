# Scopes

When you reference a variable, Python looks for it in this order:
1. current enclosing (function , class)
2. the next outer enclosing scope until the global scope, including imported modules
3. built-in module names

```python title=main.py showLineNumbers
import sys # built-in module name

from current_module import baz
# global_scope

def foo():
    # bar_outer_enclosing

    def bar():
        # current_enclosing        
        baz()
```

## Rewrite outer enclosing variable

You cannot rewrite an outer enclosing variable without the special keywords `nonlocal` and `global`.

```python title=main.py showLineNumbers
def foo():
  x = 1

  def bar():
    # highlight-next-line
    x = 2
    print('bar:' ,x)
  
  bar()
  print('foo:', x)
```

<details>

<summary> output </summary>

```bash title= showLineNumbers
bar: 2
foo: 1
```

</details>

The statement `x = 2` at line 5 would not change the value of `x` at line 2.

Python interprets `x = 2` as defining `x` in the current enclosing scope, so it has no effect on the outer scope.


### `nonlocal` keyword

To rewrite an outer enclosing variable, use the `nonlocal` keyword.

```python title=main.py showLineNumbers
def foo():
  x = 1

  def bar():
    # highlight-start
    nonlocal x 
    x = 2
    # highlight-end
    print('bar:' ,x)
  
  bar()
  print('foo:', x)
```

<details>

<summary> output </summary>

```bash title= showLineNumbers
bar: 2
foo: 2
```

</details>

### `global` keyword


To rewrite a global variable, use the `global` keyword.

```python title=main.py showLineNumbers
x = 0
def foo():
  x = 1
  def bar():
    global x 
    x = 2
    print('bar:' ,x)
  
  bar()
  print('foo:', x)

print('at global scope' , x)
```

<details>

<summary> output </summary>

```bash title= showLineNumbers
bar: 2
foo: 1
at global scope 2
```

</details>


<Note> 

`nonlocal` cannot be used to rewrite a global-scope variable.

```python title=main.py showLineNumbers
x = 0
def foo():
  # highlight-error-next-line
  nonlocal x
  x = 1
```
```bash title= showLineNumbers
File "/usr/src/app/class.py", line 3
    nonlocal x
    ^^^^^^^^^^
SyntaxError: no binding for nonlocal 'x' found
```

</Note>
