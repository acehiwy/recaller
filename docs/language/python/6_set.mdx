# Set

A set is an <Color color="var(--secondary-font-color)"> unordered </Color> collection with no duplicate elements.

```python title= showLineNumbers
basket = {'apple', 'orange', 'apple', 'pear', 'orange', 'banana'}

empty_set = set() 
```

## union, intersection, difference, and symmetric difference

```python title= showLineNumbers
a = set('abcdr')
b = set('acmlz')

a - b # left join. letters in a but not in b
# {'r', 'd', 'b'}

a | b # full join. letters in a or b or both
# {'a', 'c', 'r', 'd', 'b', 'm', 'z', 'l'}

a & b # inner join. letters in both a and b
# {'a', 'c'}

a ^ b # outer join. letters in a or b but not both
# {'r', 'd', 'b', 'm', 'z', 'l'}
```

## Set comprehensions

```python title= showLineNumbers
a = {x for x in 'abracadabra' if x not in 'abc'}
# {'r', 'd'}
```
