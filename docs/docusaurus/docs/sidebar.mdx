# Sidebar

Creating a sidebar is useful to:

- Group multiple related documents
- Display a sidebar on each of those documents
- Provide paginated navigation, with next/previous button

To use sidebars on your Docusaurus site:

1. Define a file that exports a dictionary of sidebar objects.
1. Pass this object into the `@docusaurus/plugin-docs` plugin directly or via `@docusaurus/preset-classic`.

```javascript title=docusaurus.config.js showLineNumbers
module.exports = {
  presets: [
    [
      "@docusaurus/preset-classic",
      {
        docs: {
          // highlight-next-line
          sidebarPath: require.resolve("./sidebars.js"),
        },
      },
    ],
  ],
};
```

## Autogenerated Sidebar

Docusaurus can create a sidebar automatically from your filesystem structure:
each <span style={{ color:'var(--secondary-font-color)' }}> folder </span> creates a <span style={{ color:'var(--secondary-font-color)' }}> sidebar category </span>,
and each <span style={{ color:'var(--secondary-font-color)' }}> file </span> creates a <span style={{ color:'var(--secondary-font-color)' }}> doc link </span>.

```typescript title= showLineNumbers
type SidebarItemAutogenerated = {
  type: "autogenerated";
  dirName: string; // Source folder to generate the sidebar slice from (relative to docs)
};
```
