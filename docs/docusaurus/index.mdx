# Docusaurus

Docusaurus is a <Color color="var(--secondary-font-color)"> static-site generator </Color>.
It builds a single-page application with fast client-side navigation, <Color color="var(--secondary-font-color)"> leveraging </Color> the full power of <Color color="var(--secondary-font-color)"> React </Color> to make your site interactive.

It provides <Color color="var(--secondary-font-color)"> out-of-the-box documentation feature </Color>s but can be used to create any kind of site

## Features

- Build with React
- Pluggable
  - Bootstrap your site with a basic template, advance and other community plugin
- SEO friendly
- Powered by MDX
- Search : Your full site is seachable
- Document Versioning
- i18n
