# Rebase

```bash title= showLineNumbers
A---B---C---F---G    
         \             
          D---E---H---I (HEAD)
```

## `onto` option

```bash title= showLineNumbers
git rebase --onto <new parent> <old parent> <until commit>
```

Example: extract all commit after `D` to the new parent `F`. 
```bash title= showLineNumbers
A---B---C---F---G    
            \             
              E---H---I (HEAD)
```
```bash title= showLineNumbers
git rebase --onto F D
```
<Admonition type="note" title="note"> 

It delete `D`

</Admonition>


Example: extract all commit after `D` until `H` to the new parent `F`. 
```bash title= showLineNumbers
A---B---C---F---G    
            \             
              E---H (HEAD)
```
```bash title= showLineNumbers
git rebase --onto F D H
```
