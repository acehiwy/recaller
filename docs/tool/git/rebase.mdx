## `onto` option

```bash lines
git rebase --onto <new parent> <old parent> <until commit>
```

Example: extract all commits after `D` to the new parent `F`.

```bash lines
A---B---C---F---G
            \
              E---H---I (HEAD)
```

```bash lines
git rebase --onto F D
```

<Callout type="note">

It deletes `D`.

</Callout>

Example: extract all commits after `D` until `H` to the new parent `F`.

```bash lines
A---B---C---F---G
            \
              E---H (HEAD)
```

```bash lines
git rebase --onto F D H
```
