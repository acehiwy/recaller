## Requirements

- Node.js version >= 16.14

## Create Docusaurus Project

The easiest way to install Docusaurus is to use the command line tool.

You can run this command anywhere in a new <span style={{color: "var(--secondary-font-color)"}}> empty repository </span>
or within an <span style={{color: "var(--secondary-font-color)"}}> existing repository </span>

It will create a new directory containing the scaffolded files.

``` bash
npx create-docusaurus@latest my-website classic
```

<Note>

`npx create-docusaurus@latest <name of project> <theme name> [ --typescript ]`

</Note>

#### or use these command

<Tabs>
<Tab title="npm">

``` bash
  npm init docusaurus
```

</Tab>
<Tab title="yarn">

``` bash
  yarn create docusaurus
```

</Tab>
<Tab title="pnpm">

``` bash
  pnpm create docusaurus
```

</Tab>
</Tabs>

<Tip>

Run npx create-docusaurus@latest --help, or check out its [API docs](https://docusaurus.io/docs/api/misc/create-docusaurus) for more information about all available flags.

</Tip>

- `/blog/` - Contains the blog Markdown files.
  You can delete the directory if you've disabled the blog plugin,
  or you can change its name after setting the path option
- `/docs/` - Contains the Markdown files for the docs.
  You can delete the directory if you've disabled the docs plugin,
  or you can change its name after setting the path option.
- `/src/` - Non-documentation files like <span style={{color: "var(--secondary-font-color)"}}> pages </span> or <span style={{color: "var(--secondary-font-color)"}}> custom React components </span>.
- `/src/pages` - Any `jsx` `tsx` `mdx` file within this directory will be converted into a website page.
- `/static/` - Static directory. Any contents inside here will be copied into the root of the final build directory
- `/docusaurus.config.js` - A config file containing the site configuration.
- `/sidebars.js` - Used by the documentation to specify the <span style={{color: "var(--primary-font-color)"}}> order </span> <span style={{color: "var(--secondary-font-color)"}}> of documents in the sidebar </span>

## Running Dev Server

``` bash
npm run start
```

By default, a browser window will open at [http://localhost:3000](http://localhost:3000).

## Build

``` bash
npm run build
```

Contents will be generated within the `/build` directory, which can be copied to any static file hosting service

## Update Docusaurus Version

``` bash
yarn up @docusaurus/core@latest @docusaurus/preset-classic@latest
```
