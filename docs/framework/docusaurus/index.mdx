Docusaurus is a <span style={{color: "var(--secondary-font-color)"}}> static-site generator </span>.
It builds a single-page application with fast client-side navigation, <span style={{color: "var(--secondary-font-color)"}}> leveraging </span> the full power of <span style={{color: "var(--secondary-font-color)"}}> React </span> to make your site interactive.

It provides <span style={{color: "var(--secondary-font-color)"}}> out-of-the-box documentation feature </span>s but can be used to create any kind of site

## Features

- Build with React
- Pluggable
  - Bootstrap your site with a basic template, advance and other community plugin
- SEO friendly
- Powered by MDX
- Search : Your full site is seachable
- Document Versioning
- i18n
