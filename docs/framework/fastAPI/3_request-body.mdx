# Request Body

Declare the request body type by using `Pydantic`'s <Color color="var(--secondary-font-color)"> `BaseModel` </Color>.

```python title= showLineNumbers
from fastapi import FastAPI
#  highlight-next-line
from pydantic import BaseModel


#  highlight-next-line
class Item(BaseModel):
    name: str
    description: str | None = None
    price: float
    tax: float | None = None


app = FastAPI()

@app.post("/items/")
async def create_item(item: Item):
    # item is an Item instance
    return item
```

<Admonition type="note" title="note"> 

Like query and path parameters, attributes with default values are not required when making a request.

</Admonition>
