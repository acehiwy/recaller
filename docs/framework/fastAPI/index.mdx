## Installation

```bash title= showLineNumbers
pip install 'fastapi[standard]'
```

<Admonition type="note" title="note"> 

`fastapi[standard]` package come with some optional dependencies. 

If u dont want them just use `pip install fastapi`

</Admonition>

<Admonition type="info" title="info"> 

Install fastAPI without FastAPI cloud cli, `pip install "fastapi[standard-no-fastapi-cloud-cli]"`

</Admonition>


## Hello world

```python title=main.py showLineNumbers
from fastapi import FastAPI

app = FastAPI()


@app.get("/")
async def root():
    return {"message": "Hello World"}
```

Run server:
```bash title= showLineNumbers
fastapi dev main.py
```


## Defind endpoint

use `<FastAPI instance>.<HTTP method name>()` decorator on function

- `@app.get()`
- `@app.post()`
- `@app.put()`
- `@app.delete()`
- `@app.options()`
- `@app.head()`
- `@app.patch()`
- `@app.trace()`

<Admonition type="note" title="note"> 

FastAPI call this "path operation"

</Admonition>

### returning response

```python title= showLineNumbers
@app.get("/")
async def root():
    return {"message": "Hello World"}
```

returned content can be `dict`, `list` , `str`, `int` etc

FastAPI will convert them to JSON
