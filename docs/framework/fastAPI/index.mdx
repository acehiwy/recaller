## Installation

```bash title= showLineNumbers
pip install 'fastapi[standard]'
```

<Admonition type="note" title="note"> 

The `fastapi[standard]` package comes with some optional dependencies.

If you don't want them, just use `pip install fastapi`.

</Admonition>

<Admonition type="info" title="info"> 

Install FastAPI without the FastAPI Cloud CLI: `pip install "fastapi[standard-no-fastapi-cloud-cli]"`

</Admonition>


## Hello world

```python title=main.py showLineNumbers
from fastapi import FastAPI

app = FastAPI()


@app.get("/")
async def root():
    return {"message": "Hello World"}
```

Run server:
```bash title= showLineNumbers
fastapi dev main.py
```


## Define endpoints

Use the `<FastAPI instance>.<HTTP method name>()` decorator on a function.

- `@app.get()`
- `@app.post()`
- `@app.put()`
- `@app.delete()`
- `@app.options()`
- `@app.head()`
- `@app.patch()`
- `@app.trace()`

<Admonition type="note" title="note"> 

FastAPI calls this a "path operation".

</Admonition>

### Returning a response

```python title= showLineNumbers
@app.get("/")
async def root():
    return {"message": "Hello World"}
```

The returned content can be a `dict`, `list`, `str`, `int`, etc.

FastAPI will convert it to JSON.
