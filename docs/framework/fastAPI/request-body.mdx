Declare the request body type by using `Pydantic`'s <span style={{color: "var(--secondary-font-color)"}}> `BaseModel` </span>.

``` python lines highlight={2,5}
from fastapi import FastAPI
from pydantic import BaseModel


class Item(BaseModel):
    name: str
    description: str | None = None
    price: float
    tax: float | None = None


app = FastAPI()

@app.post("/items/")
async def create_item(item: Item):
    # item is an Item instance
    return item
```

<Note> 

Like query and path parameters, attributes with default values are not required when making a request.

</Note>
