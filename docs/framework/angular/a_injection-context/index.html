Injection Context

An injection context is a place where you have access to the inject function.

Valid injection context​

Class​

Within a component, service, or directive class (a class decorated with @Component, @Injectable, or @Directive)

This includes code inside the constructor block or class property initializers.

class Foo {
  private service1 = inject(service1);

  constructor() {
    const something = inject(something);
  }
}

Factory function​

A function passed to useFactory of a Provider or an @Injectable

It also includes a factory function for InjectionToken.

API that run in an injection context​

Some APIs run inside an injection context, so it is valid to use inject.

For example, router guards such as CanActivateFn.

const canActivateTeam: CanActivateFn = (route, state) => {
  return inject(someservice);
};

Manually run code block in injection context​

Wrap the code block with runInInjectionContext.

@Injectable()
export class HeroService {
  private environmentInjector = inject(EnvironmentInjector);

  someMethod() {
    runInInjectionContext(this.environmentInjector, () => {
      inject(SomeService);
    });
  }
}
note

runInInjectionContext requires an EnvironmentInjector.
