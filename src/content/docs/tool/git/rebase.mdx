## `onto` option

```bash showLineNumbers
---
import { Aside } from '@astrojs/starlight/components';
```

Example: extract all commits after `D` to the new parent `F`.
```bash showLineNumbers
A---B---C---F---G    
            \             
              E---H---I (HEAD)
```
```bash showLineNumbers
git rebase --onto F D
```
<Aside type="note" title="note"> 

It deletes `D`.

</Aside>


Example: extract all commits after `D` until `H` to the new parent `F`.
```bash showLineNumbers
A---B---C---F---G    
            \             
              E---H (HEAD)
```
```bash showLineNumbers
git rebase --onto F D H
```
