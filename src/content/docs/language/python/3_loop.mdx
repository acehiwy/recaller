## Loop on a list

```python showLineNumbers
words = ['cat', 'window', 'defenestrate']
for w in words:
    print(w, len(w))
```

Get a list item's index.

```python showLineNumbers
words = ['cat', 'window', 'defenestrate']
for index, value in enumerate(words):
    print(index, value)
```

## Loop on dict

```python showLineNumbers
users = {'Hans': 'active', 'Éléonore': 'inactive', '景太郎': 'active'}

for key, value in users.copy().items():
    # do something
```

<Aside type="caution" title="caution"> 

Modifying and iterating over the same collection should be avoided. Use a copy instead, for example `<collection>.copy()`.

</Aside>

### `break` and `continue`


```python showLineNumbers
for ... :
  # do something
  # highlight-next-line
  continue
  for ... :
    # do something
    # highlight-next-line
    break
```

The `break` statement exits the <Color color="var(--secondary-font-color)"> innermost enclosing </Color> `for` or `while` loop.

---

### `else` on Loop


If the loop <Color color="var(--secondary-font-color)"> finishes without </Color> executing the `break`, the `else` clause executes.

```python showLineNumbers
for ... :
  # do something
else:
  print("print if no break")
```

```python showLineNumbers
i = 0
while i <= 3:
  print(i)
  i = i +1 
  break
else:
  print('print if no break')

# output
# 0
```
